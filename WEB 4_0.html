Actualidad

¿Qué es?

Se trata de una web activa. Por ejemplo, hasta la fecha, el uso de buscadores es algo importante, ya que brindan información (páginas web) sobre lo que un usuario busca o escribe. Sin embargo la web 4.0 será mucho más activa y funcionará con un asistente virtual con inteligencia artificial.
