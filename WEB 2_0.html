WEB 2.0

2001

Wikipedia

El 15 de enero de 2001 se crea Wikipedia, la enciclopedia libre, como proyecto de edición abierta que pretendía agilizar y flexibilizar el desarrollo de la enciclopedia virtual.

2004

Facebook y Gmail

Mark Zuckerberg funda la plataforma Facebook, Volviéndose a la principal red social en el mundo, iniciando el auge de las redes sociales. Google anuncia su servicio de correo electrónico: Gmail

2005

AJAX y Youtube

Se acaba la tecnología Ajax, una técnica de desarrollo web que, nos permite intercambiar información entre el servidor y el cliente (un navegador web) de forma asíncrona.

2006

Amazon EC2

Se crea Amazon Elastic Compute Cloud (Amazon EC2), un servicio web que proporciona capacidad de cálculo de tamaño ajustable en la nube. está diseñado para facilitar a los desarrolladores otra cosa es la computación escalables basadas en web.
